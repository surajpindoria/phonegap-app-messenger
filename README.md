# PG Messenger
An app based on the phonegap-template-vue-f7-blank to show realtime messaging features.

# How to run:
  1. Clone:
  2. `cd` into cloned repo
  3. `npm install`
  4. `npm run dev`

# How to build:
  1. Clone:
  2. `cd` into cloned repo
  3. `npm install`
  4. `npm run build`
  5. (Optional) `phonegap serve` and launch with the PhoneGap Developer App
